# Public Grievance Platform

## 📌 Description

Public Grievance Platform is a MERN stack web application that allows users to report civic issues in their locality. Users can submit grievances, view existing complaints, and prioritize important issues through a community-driven voting system. The platform helps improve transparency and encourages faster resolution of public problems.

## 🚀 Features

* User authentication
* Raise public grievances
* View all complaints
* Upvote / Downvote system for prioritization
* Grievance status tracking
* Admin monitoring (planned)

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Styling: CSS / Tailwind
* Authentication: JWT

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/ApoorvaGupta20/Crowdsourced-Public-Grievance-Platform.git
```

2. Navigate to project folder

```
cd pgp
```

3. Install backend dependencies

```
cd backend
npm install
```

4. Install frontend dependencies

```
cd ../frontend
npm install
```

5. Run the application

Backend:

```
npm run dev
```

Frontend:

```
npm start
```
